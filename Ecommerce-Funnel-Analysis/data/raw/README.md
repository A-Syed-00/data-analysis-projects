## Raw Data 

The raw dataset that was used in this project is too large to upload on to Github. 

**Dataset:** E-Commerce Events History In Electronics Store 
**Source:** Kaggle 
**Link:** [https://www.kaggle.com/datasets/mkechinov/ecommerce-events-history-in-electronics-store]

To reproduce this analysis: 
1. Download the dataset (csv file) from Kaggle
2. Load it onto SQLite
3. Run the SQL scripts in the `/sql` folder
